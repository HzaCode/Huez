Colormap Detection:

huez.intelligence.detect_colormap_type(data, vmin=None, vmax=None, verbose=True)[source]ÔÉÅ

Automatically detect the most appropriate colormap type for data.

This function analyzes data characteristics to recommend: - ‚Äúsequential‚Äù: Data with a natural ordering from low to high (e.g., 0-100) - ‚Äúdiverging‚Äù: Data with a meaningful center point (e.g., correlation -1 to 1) - ‚Äúcyclic‚Äù: Data that wraps around (e.g., angles 0-360)

Parameters:

  • data (ndarray | list) ‚Äì Input data as numpy array or list

  • vmin (float | None) ‚Äì Optional minimum value for range analysis

  • vmax (float | None) ‚Äì Optional maximum value for range analysis

  • verbose (bool) ‚Äì If True, print detection reasoning

Returns:

‚Äúsequential‚Äù, ‚Äúdiverging‚Äù, or ‚Äúcyclic‚Äù

Return type:

String indicating colormap type

Examples

>>> import numpy as np
>>> # Correlation matrix (-1 to 1)
>>> data = np.array([[1.0, 0.8, -0.6], [0.8, 1.0, -0.4], [-0.6, -0.4, 1.0]])
>>> detect_colormap_type(data)
'diverging'

>>> # Positive-only data
>>> data = np.random.rand(10, 10) * 100
>>> detect_colormap_type(data)
'sequential'
